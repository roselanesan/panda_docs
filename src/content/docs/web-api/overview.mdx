## Overview

All API endpoints require authentication and are designed for backend scripts, bots, and dashboard automation.

## Available APIs

### Dashboard API

Complete REST API for:

- Key generation and lookup
- Key deletion and HWID reset
- Blacklist management
- Keyless access management
- Service analytics and execution tracking
- Webhook sends

## Authentication

<Aside type="note" title="Required Header">
Include your service API key in every request:

```http
X-API-Key: your-api-key-here
```

You can find this key in your service dashboard settings.
</Aside>

## Base URL

```text
https://new.pandadevelopment.net/api/v1
```

## Quick Request Template

```http
GET /keys/api/service/status
X-API-Key: your-api-key-here
```
