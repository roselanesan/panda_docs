## Overview

```http
X-API-Key: your-api-key-here
```

Base URL:

```text
https://new.pandadevelopment.net/api/v1
```

## Key Management

<SectionTag icon={KeyRound} label="Keys" />

### Generate Keys

`POST /keys/api/generate`

```json
{
  "count": 5,
  "expirationType": "byDays",
  "expirationDays": 30,
  "isPremium": false
}
```

Parameters:

- `count` (integer, `1-100`)
- `expirationType` (`byDays` | `byDate` | `lifetime`)
- `expirationDays` (integer, required for `byDays`)
- `expirationDate` (ISO string, required for `byDate`)
- `isPremium` (boolean, optional, default `false`)

Response (`201`):

```json
{
  "success": true,
  "message": "5 keys generated successfully",
  "data": {
    "keys": [
      {
        "id": "clx...",
        "value": "PANDA-XXXX-XXXX-XXXX-XXXX",
        "isPremium": false,
        "expiresAt": "2026-03-08T...",
        "createdAt": "2026-02-06T..."
      }
    ]
  }
}
```

### Lookup Key

`GET /keys/api/key?key=PANDA-XXXX-XXXX-XXXX-XXXX`

```json
{
  "success": true,
  "data": {
    "key": {
      "id": "clx...",
      "value": "PANDA-XXXX-XXXX-XXXX-XXXX",
      "hwid": "ABC123DEF456",
      "isPremium": false,
      "expiresAt": "2026-03-08T...",
      "createdAt": "2026-02-06T...",
      "boundAt": "2026-02-06T...",
      "lastUsedAt": "2026-02-06T..."
    }
  }
}
```

### Delete Key

`DELETE /keys/api/key`

```json
{
  "key": "PANDA-XXXX-XXXX-XXXX-XXXX"
}
```

```json

  "message": "Key deleted successfully"
}
```

## Discord Bot Example

<SectionTag icon={Bot} label="Node.js Example" />

```javascript
const axios = require('axios');

const API_BASE = 'https://new.pandadevelopment.net/api/v1';
const API_KEY = 'your-api-key-here';
const headers = { 'X-API-Key': API_KEY };

async function generateKey(days, isPremium) {
    const response = await axios.post(`${API_BASE}/keys/api/generate`, {
        count: 1,
        expirationType: 'byDays',
        expirationDays: days,
        isPremium: isPremium
    }, { headers });
    return response.data.data.keys[0];
}

async function lookupKey(keyValue) {
    const response = await axios.get(
        `${API_BASE}/keys/api/key?key=${keyValue}`,
        { headers }
    );
    return response.data.data.key;
}

async function deleteKey(keyValue) {
    const response = await axios.delete(`${API_BASE}/keys/api/key`, {
        headers,
        data: { key: keyValue }
    });
    return response.data;
}

async function blacklistHwid(hwid, reason) {
    const response = await axios.post(`${API_BASE}/keys/api/blacklist`, {
        hwid, reason
    }, { headers });
    return response.data;
}

async function getStats() {
    const response = await axios.get(
        `${API_BASE}/keys/api/service/status`,
        { headers }
    );
    return response.data.data;
}

client.on('messageCreate', async (message) => {
    const args = message.content.split(' ');

    if (args[0] === '!genkey') {
        const days = parseInt(args[1]) || 7;
        const key = await generateKey(days, false);
        message.reply(`Generated: ||${key.value}|| (expires: ${key.expiresAt})`);
    }

    if (args[0] === '!lookup') {
        const key = await lookupKey(args[1]);
        message.reply(`Key: ${key.value}\nHWID: ${key.hwid || 'Unbound'}\nPremium: ${key.isPremium}\nExpires: ${key.expiresAt}`);
    }

    if (args[0] === '!stats') {
        const stats = await getStats();
        message.reply(`Active: ${stats.activeKeys} | Generated: ${stats.generatedKeys} | Executions: ${stats.totalExecutions}`);
    }

    if (args[0] === '!blacklist') {
        const result = await blacklistHwid(args[1], args.slice(2).join(' ') || 'No reason');
        message.reply(result.success ? 'User blacklisted!' : 'Failed');
    }
});
```

## Webhooks

Setup:

1. Go to service `Settings -> General`
2. Enable Discord Webhook
3. Paste webhook URL
4. Click Test
5. Save settings

Example payload (`key.validated` style event):

```json
{
  "username": "Your Service Name",
  "avatar_url": "https://your-service-logo.png",
```

## Rate Limits

<SectionTag icon={Timer} label="Rate Limits" />

- General API: `30 requests/minute`
- Execution tracking: `60 requests/minute`
- Webhook send: `5 requests/minute`

<Aside type="caution" title="429 Responses">
If you exceed the endpoint limit window, the API returns HTTP `429`.
</Aside>
